# Mini ERP Export Trading - Backend

This is the NestJS Backend application for the Mini ERP Export Trading system.

## Tech Stack
- **Framework:** NestJS v11
- **Language:** TypeScript
- **Database:** PostgreSQL (via TypeORM)
- **Queue/Cache:** Redis, BullMQ
- **Auth:** Passport-JWT

## Development

```bash
npm install
npm run start:dev
```

For more details on architecture and project structure, please refer to the [Root README](../README.md).
