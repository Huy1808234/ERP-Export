# Mini ERP Export Trading - Frontend

This is the Next.js Frontend application for the Mini ERP Export Trading system.

## Tech Stack
- **Framework:** Next.js v16+ (App Router)
- **Language:** TypeScript
- **UI Library:** Ant Design v6, Tailwind CSS v4
- **State/Auth:** React Hooks, NextAuth v5 beta

## Development

```bash
npm install
npm run dev
```

For more details on architecture and project structure, please refer to the [Root README](../README.md) and [ARCHITECTURE.md](./ARCHITECTURE.md).
