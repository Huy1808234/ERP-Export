# Design System Rule

<RULE[design_system]>
Luôn luôn đọc file `DESIGN.md` nằm ở thư mục gốc của dự án trước khi tạo mới hoặc chỉnh sửa giao diện (Frontend) để đảm bảo tuân thủ thiết kế chung (màu sắc, spacing, typography, component styling...).
Tuyệt đối không tự ý dùng các giá trị màu hoặc spacing ngoài bảng token đã được định nghĩa trong `DESIGN.md` trừ khi được yêu cầu rõ ràng.
</RULE[design_system]>
